# 遥感领域专门方向

## 目标

将 MPLSeg 的 magnitude/phase 解耦思想、ChordEdit 的低能量最优传输框架，以及可学习频域流形结构，专门应用于**高分辨率遥感图像理解任务**，包括遥感目标检测、小目标检测、变化检测、场景分类、以及遥感图像-文本对齐。

## 为什么遥感是一个独立方向

遥感图像具有与普通自然图像截然不同的特性：

1. **尺度极端**：目标尺寸跨度大（从几像素的小船/车辆到占据整幅图像的机场/港口）；
2. **方向任意**：目标朝向不固定，需要旋转不变特征；
3. **背景复杂**：地表纹理、云层、阴影、大气退化等带来强噪声；
4. **频域信息丰富**：地物在频域上具有显著的多尺度周期性结构（农田纹理、城市道路网格、海岸线等）；
5. **数据稀缺且标注昂贵**：公开高质量遥感数据集有限。

这些特性使得频域方法、流形学习方法、以及 OT 对齐在遥感上具有独特优势。

## 核心研究问题

1. **遥感小目标检测的频域增强**： magnitude/phase 解耦能否提升小目标 AP？
2. **旋转不变性**：相位谱对旋转的响应是否具有可学习的等变结构？
3. **多尺度频域流形**：遥感地物的多尺度周期性如何编码到流形坐标中？
4. **对抗防御**：针对遥感模型的物理对抗补丁（如地面标志、车体贴纸）防御。
5. **遥感图像-文本对齐**：用 OT 距离对齐遥感图像与地理文本描述。

## 预期产出

- 在 NWPU VHR-10 / VisDrone 上的频域增强检测 baseline；
- 遥感专用的 `RemoteSensingAFM` / `RemoteSensingManifold` 模块；
- 技术报告：频域方法在遥感任务上的有效性分析。
