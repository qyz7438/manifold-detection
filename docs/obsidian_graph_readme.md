# Obsidian Graph

The Obsidian vault lives in `obsidian/`.

Open `obsidian/RLIimage Map.md` as the entry point. The vault is intentionally lightweight and text-first:

- method notes link AFM, RLVR, DPO, and segmentation,
- version notes map old round numbers to new version ids,
- experiment notes link active clean runs and polluted historical runs,
- architecture notes link the migration plan and canonical runner.

The vault should be used for navigation and reasoning, not as the source of truth for metrics. Metrics remain in `runs/` and the future experiment registry.
